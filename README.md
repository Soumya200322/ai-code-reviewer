# 🤖 AI Code Reviewer

An AI-powered Python Code Reviewer built with **Streamlit** that analyzes Python code quality using **flake8**, **Black**, and **Radon**. The application helps developers identify coding style issues, automatically format code, measure code complexity, generate improvement suggestions, and download an analysis report.

---

## 🚀 Features

- Upload Python (.py) files
- Paste Python code directly
- Detect PEP 8 style violations using flake8
- Automatically format code using Black
- Measure cyclomatic complexity using Radon
- Generate code improvement suggestions
- Display formatted code
- Download code analysis report

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-code-reviewer.git
```

Go to the project folder

```bash
cd ai-code-reviewer
```

Install the dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will start at

```
http://localhost:8501
```

7. Download the analysis report.
